# Overview

Termix Desktop is the connector app to your main Termix instance. This repository does not contain the Electron source code, its only purpose is for versioning and installing.

# Planned Features

See [Projects](https://github.com/orgs/Termix-SSH/projects/2) for all planned features. If you are looking to contribute, see [Contributing](https://github.com/Termix-SSH/Mobile/blob/main/CONTRIBUTING.md).

# Installation

Termix Desktop is available to download [here](https://docs.termix.site/install).

# Support

If you need help or want to request a feature with Termix Desktop, visit the [Issues](https://github.com/Termix-SSH/Support/issues) page, log in, and press `New Issue`.
Please be as detailed as possible in your issue, preferably written in English. You can also join the [Discord](https://discord.gg/jVQGdvHDrf) server and visit the support
channel, however, response times may be longer.
